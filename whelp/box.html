Valid Text entry boxes command

You type the name of your object or part of it using the * and ? keys; TAB will attempt to autocomplete your object or class name:
  • * in the search string will match anything; i.e. "S*" will match all names beginning with 'S', "*S*" will match all names containing an 'S', "*a*b" all names containing an 'A' and ending with a 'B'.
  • ? in the template will match any single character, e.g. "???" will match all names that are exactly three characters long.
